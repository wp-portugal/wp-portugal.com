== Description ==

Outputs a list of all companies that have submitted jobs with links to their listings and profile.

= Where can I use this? =

Astoundify has released the first fully integrated WP Job Manager theme. Check out ["Jobify"](http://themeforest.net/item/jobify-job-board-wordpress-theme/5247604?ref=Astoundify)

== Installation ==

1. Install and Activate
2. Add [job_manager_companies] shortcode to any page.

== Frequently Asked Questions ==

= How can I change what shows on the company profile page? =

Create a `single-company.php` template in your child theme and add any content you desire.
