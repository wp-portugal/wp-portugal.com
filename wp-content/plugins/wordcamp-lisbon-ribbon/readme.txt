== Description ==

WordCamp Lisboa 2012 Ribbon. Core foundation from David Gwyer's Plugin Options Starter Kit.
This plugin includes 4 ribbons from which you can choose the one that better suits your site.

The ribbons are as following:
	2 x Bright Yellow Ribbon (left and right variants)
	2 x Dark Grey Ribbon (left and right variants)

Visit http://2012.lisboa.wordcamp.org for more details.	
	
== Installation ==

1. Go to Plugins => Add New.
2. Enter 'WordCamp Lisboa Ribbon' (without quotes) in the search box and click 'Search Plugins'.
3. In the list of relevant Plugins click the 'Install' link for 'WordCamp Lisboa Ribbon' on the right hand side of the page.
4. Click the 'Install Now' button on the popup page.
5. Click 'Activate Plugin' to finish installation.
6. Visit the 'Settings' page for this plugin to change the ribbon (optional).
7. Enjoy and share it with others!
