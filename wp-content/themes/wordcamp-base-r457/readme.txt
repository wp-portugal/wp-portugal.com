===============================================================================
WordCamp Base is a flexible theme designed for WordCamps.
===============================================================================


FEATURES
===============================================================================
 * Layout customization
		Customize the layout of up to five rows of widget areas.
 * Flexible grids
		Choose between 960px and 720px grids.
 * Theme styles
		Choose between two default styles.
===============================================================================


ROADMAP
===============================================================================
 * Child themes
		Convert the theme styles into child themes.
 * Custom colors
		Allow the user to choose several highlight colors for their theme.
 * Improve layout customization
		Front-page only rows
		Named widget areas
===============================================================================
